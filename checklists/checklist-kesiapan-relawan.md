# Checklist Kesiapan Relawan

## Ringkasan

Checklist ini membantu kita - relawan baru IM Cell PMI - memastikan semua sudah siap sebelum mulai bertugas. Centang satu per satu di awal penugasan.

## Akses dan Perangkat

- [ ] Laptop/PC tersedia dan berfungsi
- [ ] Aplikasi perkantoran (Excel/LibreOffice, Word, PowerPoint) terinstal
- [ ] Akun email PMI/relawan aktif dan bisa diakses
- [ ] Koneksi internet di posko sudah teruji
- [ ] Aplikasi chat tim (WhatsApp/Telegram/Signal) terpasang
- [ ] Drive bersama / folder bersama bisa diakses
- [ ] Alat tulis: pulpen, spidol, kertas HVS, binder
- [ ] Ponsel dengan paket data, kamera berfungsi

## Pemahaman Dasar

- [ ] Paham struktur IM Cell PMI dan siapa pendamping
- [ ] Paham alur data dan pengambilan data
- [ ] Paham produk-produk IM yang dipakai di posko ini
- [ ] Paham siklus pelaporan harian
- [ ] Paham classification label (PUBLIK/INTERNAL/RAPATTERBATAS)
- [ ] Sudah baca dokumen OJT di Posko
- [ ] Sudah tahu cara melaporkan masalah atau error data

## Kontak dan Komunikasi

- [ ] Nomor pendamping tersimpan di ponsel
- [ ] Nomor IM Senior tersimpan di ponsel
- [ ] Nomor Manajer Operasi/Kepala Operasi tersimpan di ponsel
- [ ] Grup chat IM Cell PMI sudah diikuti
- [ ] Paham jam kerja dan shift yang dijadwalkan
- [ ] Paham jalur eskalasi (siapa dihubungi jika pendamping tidak merespons)

## Perlengkapan Posko

- [ ] Tahu lokasi posko, pintu darurat, fasilitas umum
- [ ] Tahu jadwal listrik (jika ada pemadaman bergilir)
- [ ] Tahu lokasi genset/cadangan listrik dan cara pakai
- [ ] Tahu kontak teknisi/IT support jika perangkat bermasalah
- [ ] Tahu prosedur jika terjadi keadaan darurat (bencana susulan, kebakaran, dll)
