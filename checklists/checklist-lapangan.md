# Checklist Lapangan

## Ringkasan

Checklist ini dipakai relawan IM sebelum, saat, dan setelah turun ke lapangan. Tujuannya: tidak ada yang ketinggalan, data yang dibawa pulang layak pakai.

## Sebelum Berangkat ke Lapangan

- [ ] Tujuan dan lokasi lapangan sudah jelas
- [ ] Formulir pengumpulan data sudah dicetak/diisi (secukupnya)
- [ ] Alat tulis: pulpen, spidol, buku catatan, papan jalan (clipboard)
- [ ] Peta lokasi / titik koordinat sudah siap
- [ ] ID relawan dan rompi/seragam PMI
- [ ] Ponsel terisi penuh, power bank siap
- [ ] Aplikasi survey (KoBo/KoboCollect/ODK) sudah terinstal dan berfungsi
- [ ] Tugas dan target data sudah dikomunikasikan dengan pendamping
- [ ] Kontak person di lokasi tujuan sudah diketahui
- [ ] Logistik pribadi: air minum, masker, perlengkapan lapangan

## Saat di Lapangan

- [ ] Perkenalkan diri sebagai relawan IM PMI
- [ ] Jelaskan tujuan pengambilan data secara singkat
- [ ] Isi formulir/data dengan lengkap - jangan tunda
- [ ] Foto lokasi/objek jika relevan (minta izin)
- [ ] Konfirmasi ulang data ke responden/narasumber
- [ ] Catat kontak narasumber untuk klarifikasi lanjutan
- [ ] Jaga sikap - kita mewakili PMI
- [ ] Jika ada temuan di luar rencana, catat terpisah
- [ ] Laporkan perkembangan ke pendamping secara berkala
- [ ] Pastikan data aman (foto cadangan, simpan di cloud jika bisa)

## Setelah Kembali ke Posko

- [ ] Bersihkan data: serahkan form fisik ke pendamping
- [ ] Unggah data digital ke folder yang ditentukan
- [ ] Sampaikan temuan penting/luar biasa secara lisan
- [ ] Kembalikan perlengkapan posko yang dipinjam
- [ ] Isi catatan lapangan harian
- [ ] Konfirmasi ke pendamping bahwa tugas lapangan selesai
