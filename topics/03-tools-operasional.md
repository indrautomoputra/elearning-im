# Panduan Tools Operasional

IM Cell PMI menggunakan 4 alat utama. Masing-masing punya fungsi spesifik dalam alur kerja data: dari pengumpulan hingga penyajian.

---

## Empat Alat Utama

| Tools | Fungsi | Dipakai Saat |
|-------|--------|-------------|
| [KoboToolbox](tools-kobotoolbox.html) | Membuat form & kumpulkan data lapangan | Assessment, survei, monitoring |
| [Excel](tools-excel.html) | Olah data tabular, matching, ringkasan | Setiap hari - cleaning, pivot, lookup |
| [QGIS](tools-qgis.html) | Peta & analisis spasial | Situasi, sebaran, buffer analysis |
| [Power BI](tools-powerbi.html) | Dashboard interaktif | Presentasi, command center, update harian |

---

## Alur Tools dalam Siklus IM

1. **KoboToolbox** data masuk dari lapangan
2. **Excel** data dibersihkan dan diolah
3. **QGIS** data spasial dipetakan
4. **Power BI** data divisualisasikan dalam dashboard

Kamu tidak harus kuasai semua tools sekaligus. Pelajari bertahap sesuai tugas di posko.

---

## Triase Data (Sebelum Gunakan Tools)

Sebelum ngolah, kamu harus triase dulu tentukan data mana yang prioritas:

1. Cek **relevansi** apakah data terkait operasi saat ini?
2. Cek **aktualitas** kapan data terakhir diperbarui?
3. Cek **kredibilitas sumber** siapa pengumpul data?
4. Cek **kelengkapan** apakah data mencakup area yang diperlukan?

Prioritas 1: olah segera. Prioritas 2: olah setelah prioritas 1 selesai. Prioritas 3: simpan sebagai referensi.

---

## Catatan untuk Pendamping

Relawan tidak harus kuasai semua tools sekaligus. Ajarkan bertahap:
1. Minggu 1: triase + Excel
2. Minggu 2: KoboToolbox
3. Minggu 3: QGIS atau Power BI (salah satu dulu)

Beri tugas kecil berdasarkan tools yang paling dipakai di posko saat itu.

---

## Ringkasan Cepat

- Triase dulu sebelum ngolah data: prioritas 1 dulu
- Klik masing-masing tools di atas untuk panduan lengkap
- Satu dashboard = satu tujuan. Jangan campur aduk informasi
