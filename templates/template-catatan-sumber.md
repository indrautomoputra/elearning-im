# Template Catatan Sumber

## Ringkasan

Format standar untuk mencatat sumber data. Setiap angka dalam produk IM harus bisa dilacak kembali ke sumbernya. Catatan sumber adalah bukti bahwa data kita bisa dipertanggungjawabkan.

## Format Template

| Field | Deskripsi |
|-------|-----------|
| **Tanggal Data** | Kapan data diambil/dikumpulkan |
| **Nama Data** | Nama atau jenis data (misal: jumlah pengungsi Kecamatan A) |
| **Sumber** | Nama, instansi, kontak pemberi data |
| **Metode** | Cara data diperoleh: survey, wawancara, dokumen resmi, media |
| **Kredibilitas** | Tingkat kepercayaan: Tinggi / Sedang / Rendah (dengan alasan) |
| **Catatan** | Informasi tambahan, nomor dokumen, link file |

## Contoh Pengisian

| Tanggal Data | Nama Data | Sumber | Metode | Kredibilitas | Catatan |
|-------------|-----------|--------|--------|-------------|---------|
| 10/6 | Data KK Kecamatan Menteng | Bapak RT 04, 0821-xxx | Wawancara langsung | Tinggi - lurah setempat | Data diperbarui tiap Sabtu |
| 10/6 | Jumlah pos kesehatan | Puskesmas Kecamatan | Dokumen resmi | Tinggi - dokumen resmi | Surat No. 440/XY/2026 |
| 11/6 | Kondisi akses jalan | Relawan lapangan | Laporan langsung | Sedang - visual确认 (verifikasi) | Foto kondisi jalan menyusul |
| 12/6 | Data populasi lansia | BPS 2025 - publikasi | Data sekunder | Tinggi - sumber resmi | Tautan: tinyurl.com/xxx |

## Tips Mencatat Sumber

- **Catat segera** - Jangan menunda, detail akan lupa
- **Cetak/foto dokumen** - Simpan bukti fisik data sekunder
- **Konfirmasi kontak** - Pastikan nomor narasumber aktif
- **Bedakan fakta dan opini** - Catat jika informasi adalah perkiraan atau opini narasumber
- **Ragu? Catat** - Jika kredibilitas diragukan, tulis saja. Tim akan verifikasi
- **Standarisasi** - Pakai format template ini, jangan catat asal-asalan
