# OJT IM Cell PMI

Semi-LMS berbasis GitHub Pages untuk On Job Training (OJT) Information Management Cell PMI.

Situs ini dibuat untuk membantu relawan lokal PMI belajar sambil bertugas di lokasi bencana. Kontennya disusun agar bisa dipakai fleksibel, tidak bergantung pada jadwal kelas, dan tetap dekat dengan kondisi kerja posko.

## Tujuan

Semi-LMS ini membantu relawan untuk:

- memahami peran IM di posko,
- mengikuti alur kerja IM Cell,
- mengelola data dan informasi operasi,
- membuat atau membantu mengecek produk IM,
- menjalankan tugas OJT secara bertahap,
- tahu kapan harus eskalasi ke IM senior atau koordinator operasi.
